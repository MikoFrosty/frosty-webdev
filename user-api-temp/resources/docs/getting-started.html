Getting Started

This guide will help you get started with the User API.

The User API is a RESTful API that allows you to create, read, update, and delete users.

Create a User

To create a user, you must send a POST request to the /users endpoint.

The request body must contain a JSON object with the following properties:

  • username - The username of the user.
  • password - The password of the user.
  • email - The email of the user.

The response will contain a JSON object with the following properties:

  • id - The unique identifier of the user.
  • username - The username of the user.
  • email - The email of the user.
  • created_at - The date and time the user was created.
  • updated_at - The date and time the user was last updated.

Read a User

To read a user, you must send a GET request to the /users/:id endpoint.

The response will contain a JSON object with the following properties:

  • id - The unique identifier of the user.
  • username - The username of the user.
  • email - The email of the user.
  • created_at - The date and time the user was created.
  • updated_at - The date and time the user was last updated.
