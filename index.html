Frosty's Web Development Resources

Introduction

This is a collection of resources that I have found useful in my web development journey.

My hope is that this list will grow over time to become a useful tool for anyone who is interested in web development, whether it be as a hobby or professionaly.

Color

  • Coolors.co

    - Create custom color palettes or search through lists of suggested and trending palettes.
    - Allows for palettes to be saved for later after creating a free acount.
    - Very intuitive and easy to use.

  • BrandColors.net

    - An exhaustive collection of color palettes for most major brands.

  • Color.cloudflare.design

    - A fast way to demo your color palette to test for contrast on different designs, fonts, and icons.

Font

  • Google Fonts

    - A great resource for finding fonts that you can use in your projects.
    - Allows for fonts to be saved for later.
    - Very intuitive and easy to use.

  • Line Awesome

    - Font level icons that you can use in your projects.
    - Easy CSS integration.

  • Ultimate Google Font Pairings

    - An article that shows you some great font pairings ideas.

Image

  • Unsplash

    - A great resource for finding free images.
    - Allows for images to be saved for later.

  • Canva

    - A great resource for creating and editing images.

  • Remove.bg

    - A great resource for removing backgrounds from photos.

  • PNGfind

    - Thousands of searchable, free to use, png images.

  • Image Repeater

    - A tool for creating repeating patterns.
    - Realtime feedback for cropping images just right to get a perfect repeating pattern.

  • Devicon.dev

    - A great resource for icons and logos for skills (useful for portfolio).
    - Easy CSS integration.

  • Undraw

    - A great resource for illustrations that are free to use.
    - Very clean and professional looking illustrations.

  • Lottiefiles

    - A great resource for animations that are free to use.
    - Very clean and professional looking animations.

Layout & Design

  • Uisual.com

    - A small selection of basic free templates that look very modern and clean.

  • Mobile/Desktop Website Font Size Guidelines

    - An interesting article on best practices for font sizes with special regard for responsiveness.

  • M3.Material.io

    - Google's latest design specification standards for how devices and apps should look and operate.

HTML/CSS

  • dev.to/...

    - Article on tricks for building responsive websites without media queries.

  • W3Schools.com/howto

    - A nice collection of basic how to's for lots of common web features.
    - A great "jumping off point" for quickly implementing and building upon new features.

  • Codepip.com/games

    - A small collection of browser games that help for learning HTML, CSS, JavaScript, and more.
    - Only 2 free games available (Flexbox Froggy & Grid Garden), but they are really good for learning those CSS concepts.

  • CSS-Tricks.com

    - A great all-in-one CSS grid turorial and reference page.

  • Keyframes.app

    - A great resource for creating and editing keyframes (animations), and for quickly getting the right shadows.
    - Easy CSS integration.

  • bennettfeely.com/clippy

    - Easily create css clip-paths without realtime feedback and instant code.

Scripting/Coding

  • Awesome JavaScript

    - A list of (browser side) JavaScript libraries, frameworks, and more.

  • Playcode.io

    - quickly test out small snipets of code for easy testing and realtime feedback.

  • Codepen.io

    - Create, share, and tweak small to medium sized code snippets.
    - Allows for code snippets to be saved for later.

  • Build Your Own X

    - A collection of sites for learning how to build almost anything using various languages.
    - Very useful for inspiration and ideas for your own projects.
    - Examples include building your own text editor or a simple game, to something even loftier like your own programming language.
    - Each project has available resources for learning using various languages, and many can be done using only JavaScript.

Other

  • MixedAnalytics.com/blog/

    - A list of free APIs that are actually free and open.

  • Readme.so

    - Easily and intuitively create readme files for your GitHub repositories without knowing Markdown.

  • maenterawasi.com/...

    - A tutorial on how to create a simple web scraper using NodeJS, Axios, and Cheerio.

  • up-for-grabs.net

    - A collection of Github repositories that have open issues for new contributors.
    - Useful for learning how to contribute to open source projects, and get your feet 'wet' by doing so.
    - Note that GitHub itself has a similar collection of repositories under a section called "Goop first issue."

  • freesound.org

    - A collection of free sound effects.
    - Easy and quick to browse and download, but requires free account.
    - Good selection of sounds

  • freeconvert.com

    - A great, free, quick, online mp4 video compressor.
    - Good to making web ready mp4 videos.

  • app.copy.ai

    - Create copy with AI.
    - Easy to use, and easy to understand, free account available that allows for a limited number of free AI generated copy each month.
    - Very interesting and actually pretty good.

  • surge.sh

    - A free, open-source, and easy to use web deployment tool.
    - Great for quickly deploying static websites right from the command line.

  • Satellite Explorer

    - A free web app that allows you to view satellite imagery around the earth.
    - Not exactly good for web development, but a very cool tool :)

  • ohshitgit.com

    - A collection of common git mistakes and how to fix them.

  • web.dev

    - Uses Lighthouse to measure and improve the quality of your web apps & sites.
    - Provides a nice breakdown of the performance, accessibility, best practices, and SEO of your site.
    - Worth noting, https://web.dev/lighthouse-seo/ is a great resource for learning about SEO.

  • material.io/design

    - Good reference for best practices in creating mobile apps regarding design, UX, & components.
    - Everything is explained in detail, and there are many examples.
